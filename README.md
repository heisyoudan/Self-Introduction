# 何暓丹 - 个人作品集网站

> 一个现代化的响应式个人作品集网站，展示Java开发、Unity VR/AR和iOS应用开发技能。

## 🌐 在线预览

**访问地址**: https://heisyoudan.github.io/Self-Introduction/docs/

## ✨ 特色功能

- 🎨 **现代化设计**: 玻璃质感UI、流光动画效果
- 📱 **完全响应式**: 完美适配桌面端、平板和手机
- 🌍 **双语支持**: 日语/中文无缝切换
- 🎯 **技能展示**: 动态进度条展示技术能力
- 📈 **项目作品集**: 详细的项目介绍和演示
- ⏰ **职业时间线**: 清晰的职业发展历程

## 🛠️ 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **样式**: 原生CSS + 现代化效果（backdrop-filter, CSS Grid/Flexbox）
- **交互**: 原生JavaScript + jQuery
- **国际化**: 自定义i18n系统
- **响应式**: Mobile-first设计
- **动画**: CSS Keyframes + Transform

## 📁 项目结构

```
docs/
```

## 🎯 关键特性

### 设计系统
- **玻璃morphism效果**: backdrop-filter实现现代玻璃质感
- **一致的设计语言**: 统一的配色方案和交互模式
- **流畅动画**: 60fps的hover动画和过渡效果

### 用户体验
- **直观导航**: 清晰的信息架构和导航结构
- **快速加载**: 优化的资源加载和渲染性能
- **无障碍支持**: 符合WCAG标准的可访问性设计

### 技术亮点
- **原生JavaScript**: 无框架依赖，轻量高效
- **模块化架构**: 可维护的代码结构
- **SEO优化**: 完善的meta标签和语义化HTML
